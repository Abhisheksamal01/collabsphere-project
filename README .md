# CollabSphere - Real-time Collaborative Project Management

Advanced full-stack project using **Spring Boot 3 + React 18 + MySQL + WebSocket**.

## Features
- JWT Authentication with RBAC
- Real-time Task Updates via WebSockets
- Kanban Board with Drag & Drop
- Analytics Dashboard with Charts
- Modern Animated UI (Framer Motion + Glassmorphism)
- File Attachments (extendable)

## Tech Stack
**Backend**: Spring Boot, JPA, Spring Security, WebSocket, JWT  
**Frontend**: React, Tailwind, Framer Motion, Recharts  
**Database**: MySQL

## Setup Instructions

### Backend
1. Import `backend` folder in IntelliJ / VS Code
2. Update `application.yml` with your MySQL credentials
3. Run `CollabsphereApplication.java`

### Frontend
```bash
cd frontend
npm install
npm run dev
```
